## 🌟 Highlights & Features

- **⚡ Blazing Fast Performance**: Built using Next.js 15 App Router with static pre-rendering and server components.
- **🎨 Premium Dark UI**: Custom-tailored dark mode aesthetics with smooth glassmorphism effects and modern typography.
- **✨ Dynamic Micro-Animations**: Smooth entry, scroll, and hover animations powered by Framer Motion.
- **🛠️ Tech Stack Showcase**: Highlighted frontend, backend, database, and tool competencies with brand accents.
- **📁 Featured Projects**: Display of full-stack projects with live demo and source code links.
- **📬 Interactive Contact Form**: Fully integrated contact form with real-time toast feedback (`react-hot-toast`).
- **📱 Fully Responsive**: Fluid, mobile-first design optimized for desktops, tablets, and mobile screens.
- **🔍 Full SEO & OpenGraph**: Rich link previews for LinkedIn, X (Twitter), WhatsApp, and search engines.

## 🚀 Getting Started

Follow these steps to run the portfolio locally on your machine:

### **1. Clone the repository**
```bash
git clone https://github.com/sarthakk20/Portfolio.git
cd Portfolio
```

### **2. Install dependencies**
```bash
npm install
```

### **3. Run the development server**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

---

## 🏗️ Production Build

To build and verify the app for production:

```bash
npm run build
npm run start
```
